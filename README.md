# Asteroid-Shooter
An asteroid shooter game implemented in Python and Pygame.


REQUIREMENTS
------------
* Python 2.x
* Pygame

(pip and setuptools will be implemented soon)

CONFIGURATION
-------------

Just run `python main.py` . 

CONTROLS
--------
I - UP

K - DOWN

J - LEFT

L - RIGHT


A - ROTATE LEFT

D - ROTATE RIGHT


SPACE - SHOOT

P - PAUSE
